# 🚀 ג'אבי הקודקוד – תכנון משחק מלא (4 רמות)

מסמך זה מציג 4 רמות עם **עלייה ברורה בקושי** ועם **עקרונות תכנות אמיתיים**: תנאים, רצף פעולות, תנאי עם "אחרת", ולולאות.

הקושי עולה בהדרגה דרך:

* מספר ההחלטות
* אורך המשימה
* רמת המופשטות
* שילוב של יותר ממושג אחד

---

# 🌱 רמה 1 – If / Then (קל מאוד)

## ⭐ מטרת הרמה

הילד מבין את עיקרון **"אם לוחצים – קורה משהו"**.

## 🎮 מה קורה ברמה

* ג'אבי עומד ליד עציץ ריק.
* הקריינית: "אם נבחר פקודה – ג'אבי יעשה אותה!"
* מופיעים 3 כפתורים גדולים:

  * קדימה 🚶‍♂️
  * קפיצה 🟰
  * לישון 😴

### דוגמה חינוכית – קוד סמוי:

```
אם לוחצים קדימה:
    ג'אבי מתקדם
אם לוחצים קפיצה:
    ג'אבי קופץ
אם לוחצים לישון:
    ג'אבי ישן
```

## 📜 תרשים זרימה

בחירת פקודה → פעולה → צליל → מדבקה → המשך.

## 🎚️ רמת קושי

* פעולה אחת בלבד
* פידבק מיידי
* לא צריך לתכנן קדימה

## 🟦 סיום

מדבקה: **"המתכנת הקטן!"**

---

# 🍃 רמה 2 – רצף פעולות (Sequence) – קל

## ⭐ מטרת הרמה

להבין שסדר הפעולות חשוב.

## 🎮 מה קורה ברמה

* ג'אבי ליד זרע.
* הקריינית: "כדי להגיע לזרע – צריך ללכת ברצף!"
* מופיע שביל של 3 משבצות.
* הילד צריך ללחוץ **קדימה** פעמיים.

### שגיאה חיובית:

```
אם הילד לוחץ קפיצה:
    ג'אבי קופץ במקום
    הקריינית: "זה כיף – אבל לא מתקדמים!"
```

## 📜 תרשים זרימה

קדימה → קדימה → מגיע לזרע → אוסף → סיום.

## 🎚️ רמת קושי

* שני צעדים רצופים
* הילד *חייב לחשוב צעד קדימה*

## 🟦 סיום

העציץ: **"מוכן לשתילה"**.

---

# 🌤️ רמה 3 – If / Else (בחירה נכונה) – בינוני

## ⭐ מטרת הרמה

להבין שיש **בחירות נכונות** ו**בחירות שלא מתאימות**.

## 🎮 מה קורה ברמה

* ג'אבי מחזיק זרע.
* הקריינית: "איפה שותלים זרע?"
* מופיעות 3 אפשרויות:

  1. אדמה 🌱
  2. שולחן 🪑
  3. הראש של ג'אבי 🤣

### קוד סמוי:

```
אם נבחרה אדמה:
    שתול את הזרע
אחרת:
    הצג משוב עדין ונסה שוב
```

### תגובות:

* בחירה נכונה -> אנימציית שתילה.
* בחירה מצחיקה -> ג'אבי מצחקק.

## 📜 תרשים זרימה

בחירה -> בדיקה -> (נכון → שתילה) / (שגוי → משוב) -> המשך.

## 🎚️ רמת קושי

* הילד בוחר מתוך 3 אפשרויות
* נדרש להבין "למה זה לא נכון"

## 🟦 סיום

הזרע קבור, מופיעה מדבקה: **"מסדר הפקודות"**.

---

# 🌼 רמה 4 – לולאה (Loop) – בינוני+

## ⭐ מטרת הרמה

להבין חזרתיות: פעולה שחוזרת *כמה פעמים*.

## 🎮 מה קורה ברמה

כדי שהפרח יצמח צריך:

* לתת מים **3 פעמים** 💧
* לתת שמש **פעמיים** ☀️

### קוד חינוכי סמוי:

```
עשה 3 פעמים:
    תן מים

עשה 2 פעמים:
    תן שמש
```

### איך זה נראה במשחק:

* כל לחיצה על מים מעלה מונה: 1/3, 2/3, 3/3.
* כל לחיצה על שמש: 1/2, 2/2.
* כשלולאה מסתיימת -> מתנגן צליל התקדמות.

## 🎚️ רמת קושי

* הילד צריך לספור
* להבין רמות התקדמות
* לבצע פקודות חוזרות

## 🟦 סיום

הפרח צומח → מדבקה: **"ג'אבי יודע לולאות!"**

---

# 🎉 סיכום עליית הקושי

| רמה | מושג תכנותי           | קושי    | למה הקשה יותר?              |
| --- | --------------------- | ------- | --------------------------- |
| 1   | תנאי בסיסי (If)       | קל מאוד | פעולה בודדת ללא חשיבה קדימה |
| 2   | רצף פעולות (Sequence) | קל      | הילד מתכנן 2–3 צעדים        |
| 3   | If/Else               | בינוני  | יש בחירה נכונה/שגויה        |
| 4   | לולאה (Loop)          | בינוני+ | ספירה + ביצוע חזרתי         |

* הפרח פורח 🌸
* ג'אבי מוחא כפיים.
* מוצגים כל הישגי הילד.
* מסר סופי: **"כל הכבוד! את/ה מתכנת/ת מצוין!"**
